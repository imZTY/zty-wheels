# zty-wheels
 通用中台轮子项目

## 模块介绍
```
├─account 插件模块:账号基础能力(1.0)
├─api-gateway 应用模块:api网关模块(未开始)
├─file 插件模块:文件基础能力(1.0)
├─framework 框架模块:zty的开发框架(1.0)
├─msg 插件模块:消息基础能力(未开始)
├─pay 应用模块:支付中心(建设中)
├─redis 插件模块:redis缓存基础能力(1.0)
├─task 插件模块:定时任务基础能力(未开始)
└─wx 插件模块:对接微信基础能力(未开始)
```

### pay模块

pay-web 基于开源thl-pay的示例项目抽离的模块
